Page dynamique

Ici, vous pouvez utiliser les balises 4D dans la page HTML.
Ces balises sont interprétés par 4D avant d'envoyer la page à votre navigateur.

Exemple 1

Je veux ajouter du texte dynamique sur ma page avec la balise 4DTEXT.

<!--#4DTEXT Current Time--> :

<!--#4DTEXT Current Date--> :

Exemple 2

J'ai besoin d'appeler une méthode 4D pour exécuter du code, j'utilise la balise 4DSCRIPT.

<!--#4DSCRIPT/maMethodeQuiNExistePas--> :

Attention, vous pouvez rencontrer cette erreur si vous n'avez pas coché la case "Balises HTML et URLs 4D (4DACTION...)" dans les propriétés de la méthode.

<!--#4DSCRIPT/maMethodePourLeWeb--> :

Exemple 3

J'ai besoin d'afficher le contenu d'une table.

Je dois commencer par initialiser la sélection des enregistrements à afficher en appelant une méthode 4D :

<!--#4DSCRIPT/methodeQuiInitialise--> : (Ma méthode ne retourne rien)

Ensuite, pour boucler, j'utilise les balises 4DLOOP...4DENDLOOP.

<!--#4DLOOP [PERSON]-->
  • <!--#4DTEXT [PERSON]Firstname--> <!--#4DTEXT [PERSON]Lastname--> :
<!--#4DENDLOOP-->

Conslusion

Vous pouvez manipuler un peu ces quelques balises afin de vous familiariser avec la logique.
